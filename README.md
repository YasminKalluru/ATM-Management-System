# ATM Management System

## Project Overview

The ATM Management System is a Python-based application that simulates the basic functionalities of an Automated Teller Machine (ATM).

This project is being developed step by step to learn Python programming, software engineering principles, Git & GitHub, databases, backend development, and frontend development.

## Technologies Used

- Python 3.14
- Git
- Visual Studio Code
